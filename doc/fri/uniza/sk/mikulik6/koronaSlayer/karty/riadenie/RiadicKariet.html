Class RiadicKariet

  • 
public final class RiadicKariet

    Trieda zodpovedajúca za prácu s kartami počas hry. Trieda obsahuje pole vzorov kariet, z ktorých sú 3 generované na konci levelu. Okrem iného obsahuje listy ako sú zahadzovací, ťahací balíček, hratelné karty a použité karty, ktoré sú používané počas levelu.
